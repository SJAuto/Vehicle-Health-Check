Help & Instructions

Using the Vehicle Health Check Dashboard

This dashboard allows you to manage all vehicle health checks.

Navigation

  • Dashboard: View all health checks
  • Menu: Access additional options
  • Help: View this help information

Actions

  • New Health Check: Create a new vehicle health check
  • Refresh: Update the list of health checks
  • View: View the details of a health check
  • Edit: Modify an existing health check
  • Delete: Remove a health check (admin only)
