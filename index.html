Help & Instructions

Using the Vehicle Health Check System

This system allows you to create, view, and edit vehicle health checks.

Navigation

  • Dashboard: View all health checks
  • Menu: Access additional options
  • Help: View this help information

Creating a Health Check

  1. Fill in the vehicle information
  2. Complete the health check items for each section
  3. Add any technician notes
  4. Save the health check

Tips

  • Use the navigation buttons to move between steps
  • All required fields must be completed before saving
  • You can print or save a PDF of the health check
